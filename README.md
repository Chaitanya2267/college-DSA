# college-DSA

Java Data Structures and Algorithms (DSA) Collection
This repository contains 10 fundamental Java implementations of core Data Structures and Algorithms. These scripts cover everything from basic array and matrix operations to advanced sorting techniques and linked-based memory management.

 Project Overview
The collection is designed to demonstrate efficient logic for data manipulation. Each file is a standalone implementation, making it easy to study specific concepts like memory pooling, pointer manipulation in linked lists, and time-complexity optimized sorting.
